📊 Netflix Content Insights Dashboard

🔹 Project Overview : 
The Netflix Content Insights Dashboard is a data analytics project built using Microsoft Power BI to analyze movies and TV shows available on Netflix.
This dashboard helps users explore content trends and gain insights through interactive visualizations.

🔹 Objectives : 
Analyze distribution of movies and TV shows
Understand genre and rating patterns
Identify country-wise content availability
Track release year trends
Compare Movies vs TV Shows

🔹 Key Features : 
📌 Interactive filters for better data exploration
🎬 Movies vs TV Shows comparison
📊 Genre-wise and rating-wise analysis
🌍 Country-wise content distribution
📈 Release year trend visualization

🔹 Tools & Technologies : 
Microsoft Power BI
Data Cleaning & Transformation
Data Modeling
DAX (Data Analysis Expressions)

🔹 Key Insights : 
Most content is dominated by specific genres
Movies are more in number compared to TV shows
Certain countries contribute more content
Content release has increased over the years

🔹 Outcome : 
This project demonstrates how raw data can be transformed into meaningful insights using interactive dashboards, helping users understand content trends effectively.
